### Commands

```bash
# Create virtual environment
python3 -m venv venv

# Activate virtual environment
source venv/bin/activate

# Install dependencies
pip install -r requirements.txt

# Run the server
flask run
or
FLASK_DEBUG=1 flask run
```

### How to use
```curl
curl --location 'http://127.0.0.1:{PORT}/?q=100.0'
```
